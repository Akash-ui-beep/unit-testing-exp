# Unit Testing for Frontend and Backend

## Objective
To perform unit testing on backend APIs using pytest and frontend form validation using Vitest.

## Backend Testing
- Framework: Flask
- Tool: pytest
- API tested: /add

### Command:
cd backend
pytest

## Frontend Testing
- Tool: Vitest
- Function tested: validateForm()

### Command:
cd frontend
npm install
npm test

## CI/CD
- Implemented using GitHub Actions
- Workflow automatically runs tests on push

## Output
- Backend tests passed
- Frontend tests passed
- CI pipeline successful
